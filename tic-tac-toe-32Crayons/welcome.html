Tic Tac Toe Game

Rules:

  1. There are two players in the game. First player is assigned symbol - X, Second player is assigned symbol - O

  2. Both players have different names before game begins, if same a similar prompt will notify the problem

  3. Each player can mark his position in any of the unmarked boxes in the 3X3 matrix by clicking using the mouse

  4. The first player to align 3 of his symbols in a line horizontally, vertically or diagonally wins the game

  5. If all the boxes of the matrix are filled and there is no winner then the game comes to a draw

  6. If the game needs to be reset hit reload

  7. The player whose chance is to play will be displayed below the matrix
